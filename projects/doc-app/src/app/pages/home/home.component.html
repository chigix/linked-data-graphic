Linked Data Graphic

Linked Data Graphic and Visualization Components for Angular

Installation link

Initially, this library is designed as an Angular Component and released as an npm package, for which Angular Development Environment and an npm package manager is prerequisites.

npm install @ngld/canvas

Getting Started: Thorin and Company Path link

Open the module file (e.g app.module.ts) and import the LinkedDataGraphicModule symbol from linked-data-graphic library. Then add LinkedDataGraphicModule to the @NgModule metadata's imports array.

{{ codes.installInTs }}

Prepare a simple list of linked data with nodes and relations information in your application component(e.g app.component.ts) as a property for binding into ngld-canvas component later.

{{ codes.bindingProperty }}

Add a <ngld-canvas> element, the <ngld-canvas> component selector, into your application's template, where your linked data is intended to be shown.
